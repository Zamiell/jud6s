  The Jud6s Mod

A racing mod for The Binding of Isaac: Afterbirth

Download the Isaac Racing Mods package

Or, download only the Jud6s mod

What Is It?

This is a mod that tweaks the game to be more fair for racing. It also fixes a ton of bugs that remain unpatched in the vanilla game.

This mod is commonly used while racing on SpeedRunsLive and is also chosen for many tournaments, such as Roid Rage and Balls of Steel.

Installation & Uninstallation

If you are using the Isaac Racing Mods package, it will automatically handle installing and uninstalling the mod for you. Otherwise, see the instructions below for your respective operating system.

Windows

After unzipping the zip file to a new directory, double click on the INSTALL (Windows) script. It will take care of everything for you. (This script can also be used to uninstall the mod.)

Alternatively, you can manually install the mod by copying all of the files in the "resources" directory to:
C:\Program Files (x86)\Steam\steamapps\common\The Binding of Isaac Rebirth\resources\

To go back to an unmodded game, delete everything in the resources directory except for the "packed" directory.

Mac & Linux

After unzipping the mod files, you should see a "resources" subdirectory. Install the mod by copying the contents of that directory into your Isaac resources directory. On Mac, this is typically:
/Users/[your-user-name]/Library/Application Support/Steam/SteamApps/common/the binding of isaac/The Binding Of Isaac.app/Content/Resources/resources/

Design Goals

Broadly, the Jud6s Mod has several goals, and attempts to strive a balance between them. However, certain things are prioritized. The goals are listed in order of importance:

  • to increase the chance that a skilled player will beat an unskilled player
  • to make the game more fun to play
  • to fix bugs and imperfections
  • to keep the game as "vanilla" as possible

Ruleset

The mod is meant to be played:

  • with Judas
  • on normal mode
  • with the BLCK CNDL seed ("total curse immunity")
  • on a "1001%" save file (which is a save file that has everything in the game unlocked)

If don't already have one, you can download a 1001% save file from Speedrun.com.

Mod Changes

1. Judas is tweaked:

  • Judas starts with the D6. (All other characters also start with the D6, except for Eve, Eden, and Keeper, as it isn't possible.)
  • Judas starts with half a soul heart in addition to his 1 red heart container (so that he can consistently take a devil deal).
  • Judas starts with a bomb instead of 3 coins (so that he can get Treasure Room pedestal items surrounded by rocks).

2. Devil Rooms and Angel Rooms are buffed:

  • Devil Rooms and Angel Rooms without item pedestals in them have been removed.

3. Some items with no effect are removed:

  • the Karma trinket (all Donation Machines are removed on the BLCK CNDL seed)
  • the Amnesia pill (all curses are removed on the BLCK CNDL seed)
  • the ??? pill (all curses are removed on the BLCK CNDL seed)

4. Cutscenes are removed.

5. Some useless animations are removed:

  • cowering in the fetal position at the beginning of every floor
  • jumping in a hole to the next floor
  • going up the beam of light to the Cathedral
  • entering a chest when going to The Chest or beating the game

6. Some death animations are shortened:

  • Boss death animations that are longer than 1 frame are set to 1 frame.
  • The Satan death animation is set to 30 frames.
  • Krampus, Uriel, Gabriel, and It Lives are unchanged.

7. Some bug fixes:

  • Charge animations with transformations now work.
  • Many rooms with unavoidable damage or bugs have been fixed or deleted. Full documentation can be found on the Room Changes page.

8. Lag reduction:

  • Fog has been removed to help eliminate lag on slower computers and make the game clearer to see.
  • A config.ini file has been included to disable some graphical effects, such as the water on Flooded Caves. If you already have a config.ini in your resources directory, the Isaac Racing Mods program (or the "INSTALL (Windows).bat" file) will not bother copying one over. If you don't like the way that the config.ini makes the game look, leave the file it in your resources directory and make the contents completely blank.
